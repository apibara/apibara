# Parquet Integration

**Notice**: the Parquet integration is still in the early stage and subject to
change.

The Parquet integration is used to generate datasets containing onchain data.
Parquet is the standard file format used by analytics applications such as:

 - The Pandas and Polars dataframe libraries for Python.
 - DuckDB, an in-process SQL OLAP database.


### Installation

```
apibara plugins install sink-parquet
```


### Configuration

 - `outputDir: string`: write the Parquet files to this directory.
 - `batchSize: string`: each Parquet file contains data for the specified
   number of blocks.
