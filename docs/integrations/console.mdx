# Console Integration

This integration is used to quickly debug Apibara scripts by printing the
result of the transform function to standard output.


### Installation

```
apibara plugins install sink-console
```

### Usage

Simply set the value of `sinkType` to `console`.

```ts
export cost config = {
  // your config here...
  sinkType: "console",
  sinkOptions: {
  }
}
```
